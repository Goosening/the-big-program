Detailní popis použitých technologií, architektury a optimalizačních postupů aplikovaných při vývoji webu.

1. Architektura a Technické specifikace

Projekt byl realizován pomocí čistého HTML5, CSS3 a Vanilla JavaScriptu (ES6+) bez frameworků.

  • Vícestránková architektura (MPA): Web je rozdělen do více HTML souborů, což umožňuje čistší organizaci obsahu a přesnější zacílení klíčových slov pro SEO na konkrétních podstránkách.
  • Rozložení (Layout): Využit moderní přístup CSS Flexbox pro "Sidebar" navigaci a CSS Grid pro mřížku karet (např. sekce Proč AI). Tento designový vzor je čistý a efektivně odděluje navigaci od obsahu.
  • JavaScript: Napsán v moderním formátu ES6+. Stará se o to, že analyzuje aktuální URL adresu (window.location.pathname) a podle toho dynamicky přiřazuje třídu .active správnému odkazu v postranním menu.

2. Oblasti optimalizace

Výkon (Performance)

  • Minimalizace HTTP požadavků: Místo importování externích webfontů (např. Google Fonts) jsou využity moderní systémové fonty. Toto řešení zamezuje stahování dalších souborů a text je vykreslen okamžitě bez tzv. FOUT (Flash of Unstyled Text).
  • Asynchronní načítání: JavaScript je načítán s atributem defer, takže neblokuje prvotní renderování HTML dokumentu.
  • Žádný zbytečný kód: CSS soubor má minimum pravidel a kód je připraven pro bezproblémovou minifikaci v produkci.

SEO (Optimalizace pro vyhledávače)

  • Sémantika: Web používá sémantické tagy HTML5 (<aside>, <main>, <section>, <article>) pro dokonalé pochopení struktury vyhledávacími roboty.
  • Sitemap a Robots.txt: Součástí struktury projektu je sitemap.xml udržující přehled všech podstránek a robots.txt.

Přístupnost (Accessibility - A11y)

  • Barevný kontrast v Dark Mode: Hluboké modré pozadí a téměř bílý text byly zvoleny tak, aby splňovaly nejpřísnější standardy čitelnosti WCAG, přičemž jsou velmi příjemné na oči (zamezují tzv. eye-strain).
  • Sémantická navigace: Navigační prvek používá atribut aria-label="Hlavní navigace".

UI/UX a Responzivita

  • Mobile First a plynulý přechod: Kód plynule přeskupuje postranní panel (Sidebar) do horní nebo spodní lišty na menších obrazovkách pomocí @media queries, aniž by byl potřeba složitý JavaScript pro "hamburger" menu.
  • Design: Tmavé téma (Dark theme) a minimalistická typografie dávají webu vzhled profesionální technologické dokumentace či portfolia.
